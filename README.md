# I'm Mirabelle

**Design Engineer — Design Systems**  

I love being at the intersection of design and engineering. Turning design into production-ready components, defining how they’re used through clear APIs, and building token systems for consistency across teams. Accessibility is built in from the beginning.

1. Most of my case studies live under [Mirabelle-Workspace](https://github.com/Mirabelle-Workspace) - design system experiments, prototypes, and works-in-progress where ideas get tested before they ship. 
2. Alongside it, [mirabelle-design-systems-org](https://github.com/mirabelle-design-systems-org) holds my component library work
3. [The-Wednesday-Labs](https://github.com/The-Wednesday-Labs) is the umbrella for digital tools built for animal welfare
4. [Organisme-Respire](https://github.com/Organisme-Respire) is digital infrastructure for a Montréal-based qualitative urban research nonprofit.
